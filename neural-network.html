2. Core Component Definitions

Detailed functional roles of the elements mapped in the architecture above.

Layers

Neural networks process data in sequential stages. The Input Layer ingests raw multidimensional tensors. Hidden Layers perform hierarchical feature extraction, combining simple patterns into complex abstractions. The Output Layer formats the final probability distribution.

Neurons (Nodes)

The fundamental computational units. Mathematically, a neuron calculates the weighted sum of its inputs, adds a bias term, and passes the result through an activation function to determine its output signal: y = f(∑wᵢxᵢ + b)

Weights

The learnable numerical parameters that determine the "importance" or magnitude of the signal passing between two neurons. Adjusting these weights is how the network continuously learns to map inputs to the correct outputs.

Activation Functions

Mathematical transformations applied to a neuron's output. Without them, the network collapses into a basic linear regression model. Functions like Tanh or ReLU introduce the crucial non-linearity required to carve out curved decision boundaries.

Loss Functions

The metric utilized to evaluate model performance. It quantifies the "cost" or error by calculating the distance between the model's predicted output and the ground truth label. The goal of training is to minimize this value.

Optimization Algorithms

The engine of learning. Algorithms like Gradient Descent utilize backpropagation to calculate the gradient of the loss function. The optimizer iteratively adjusts the weights in the opposite direction of the gradient to find the lowest error.

3. Playground Case Study: The Circular Dataset

Experiment Parameters

  • Dataset: Non-Linear Circle
  • Architecture: 4 Neurons → 2 Neurons
  • Activation: Tanh
  • Learning Rate: 0.03

Key Observations

1. The Necessity of Depth: Attempting to classify the circular dataset with zero hidden layers failed immediately. A model without hidden layers is strictly linear; it cannot draw the enclosed boundary required to separate a circle from an outer ring. Hidden layers are mandatory for spatial transformation.

2. The Mechanics of the 4x2 Architecture: By configuring 4 neurons in the first hidden layer, the network was able to establish four intersecting linear boundaries (functioning effectively as a bounding box). The subsequent layer of 2 neurons aggregated those boundaries, smoothing the intersections to form a continuous circular classification zone.

3. Hyperparameter Sensitivity: The learning rate of 0.03 allowed the optimization algorithm to converge smoothly. However, artificially increasing the rate caused the model to drastically overshoot the optimal weights. The gradients exploded, preventing the loss function from ever minimizing, which visually resulted in a chaotic, flickering decision boundary.
