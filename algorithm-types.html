Detailed Algorithm Classification Matrix

Algorithm Learning Type Primary Domain Brief Explanation Real-World Use Case
1. Random Forest Supervised Tabular Data Builds an "ensemble" of multiple decision trees and averages predictions to prevent overfitting. Predicting customer churn rates or loan defaults based on historical banking data.
2. Gradient Boosting (XGBoost) Supervised Tabular Data Sequentially builds decision trees, where each new tree corrects the errors of the previous ones. High-frequency financial trading and dynamic pricing models.
3. CNNs Supervised Computer Vision Uses spatial "filters" to scan images pixel-by-pixel, identifying basic features and combining them into complex patterns. Medical imaging analysis, such as detecting tumors in MRI scans.
4. YOLO Supervised Computer Vision Passes an image through a neural network a single time to instantly predict bounding boxes and object probabilities. Real-time object detection for autonomous driving systems.
5. BERT Supervised NLP Reads entire sequences of text simultaneously to understand the deep contextual meaning of words based on their surroundings. Google Search query understanding and automated sentiment analysis.
6. K-Means Clustering Unsupervised Tabular Data Partitions unlabeled data into 'K' distinct groups by calculating the spatial distance between data points. Market segmentation, grouping e-commerce customers by purchasing behavior.
7. PCA Unsupervised Tabular Data A dimensionality reduction technique that compresses datasets down to the most critical features while retaining core variance. Compressing images before applying predictive models.
8. GANs Unsupervised Generative AI Pits two neural networks against each other: a "Generator" creates fake data, and a "Discriminator" tries to catch the fakes. Generating highly realistic synthetic training data for scenarios where real data is scarce.
9. Diffusion Models Unsupervised Generative AI Progressively adds noise to an image until it is destroyed, then learns how to reverse the process to generate new images. Creating high-quality digital art or marketing assets from text prompts.
10. GPT Self-Supervised GenAI / NLP Uses self-attention mechanisms on massive text datasets to predict the most statistically probable next word in a sequence. Powering conversational agents, generating code, and automating complex writing tasks.
