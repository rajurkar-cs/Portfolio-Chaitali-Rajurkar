Detailed Algorithm Directory

A breakdown of how each algorithm functions and where it is applied in the real world.

Random Forest

Tabular Data
Supervised Learning

Builds an "ensemble" of multiple decision trees and averages their predictions to prevent the model from overfitting to the training data.

Use Case: Predicting customer churn rates or loan defaults based on historical banking data.

XGBoost

Tabular Data
Supervised Learning

Sequentially builds decision trees, where each new tree focuses specifically on correcting the mathematical errors of the previous ones.

Use Case: High-frequency financial trading and dynamic pricing models (e.g., Uber surge pricing).

CNNs

Computer Vision
Supervised Learning

Uses spatial "filters" to scan images pixel-by-pixel, identifying basic features (edges) and combining them into complex patterns (shapes).

Use Case: Medical imaging analysis, such as detecting tumors in MRI scans.

YOLO

Computer Vision
Supervised Learning

Passes an image through a neural network only a single time to instantly predict bounding boxes and object probabilities simultaneously.

Use Case: Real-time object detection for autonomous driving systems.

BERT

NLP
Supervised Learning

Reads entire sequences of text simultaneously to understand the deep contextual meaning of words based on their surroundings.

Use Case: Google Search query understanding and automated sentiment analysis.

K-Means Clustering

Tabular Data
Unsupervised Learning

Partitions unlabeled data into distinct groups by calculating the spatial distance between data points and central nodes.

Use Case: Market segmentation, grouping e-commerce customers by purchasing behavior.

PCA

Tabular Data
Unsupervised Learning

A dimensionality reduction technique that compresses datasets down to the most critical features while retaining core variance.

Use Case: Compressing high-resolution images or genetic data before applying predictive models.

GANs

Generative AI
Unsupervised Learning

Pits two neural networks against each other: a "Generator" creates fake data, and a "Discriminator" tries to catch the fakes.

Use Case: Generating highly realistic synthetic training data for scenarios where real data is scarce.

Diffusion Models

Generative AI
Unsupervised Learning

Progressively adds noise to an image until it is destroyed, then learns how to reverse the process to generate brand new images.

Use Case: Creating high-quality digital art or marketing assets from text prompts (e.g., Midjourney).

GPT

GenAI / NLP
Self-Supervised Learning

Uses self-attention mechanisms on massive text datasets to predict the most statistically probable next word in a sequence.

Use Case: Powering conversational agents, generating code, and automating complex writing tasks.
