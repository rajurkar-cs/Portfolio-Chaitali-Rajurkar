Detailed Algorithm Classification Matrix

Algorithm Learning Type Primary Domain Brief Explanation Real-World Use Case
1. Random Forest Supervised Tabular Data Builds an "ensemble" of multiple decision trees and averages predictions to prevent overfitting. Predicting customer churn rates or loan defaults based on historical banking data.
2. XGBoost
(eXtreme Gradient Boosting)		 Supervised Tabular Data Sequentially builds decision trees, where each new tree corrects the errors of the previous ones. High-frequency financial trading and dynamic pricing models.
3. CNNs
(Convolutional Neural Networks)		 Supervised Computer Vision Uses spatial "filters" to scan images pixel-by-pixel, identifying basic features and combining them into complex patterns. Medical imaging analysis, such as detecting tumors in MRI scans.
4. YOLO
(You Only Look Once)		 Supervised Computer Vision Passes an image through a neural network a single time to instantly predict bounding boxes and object probabilities. Real-time object detection for autonomous driving systems.
5. BERT
(Bidirectional Encoder Representations from Transformers)		 Supervised NLP Reads entire sequences of text simultaneously to understand the deep contextual meaning of words based on their surroundings. Google Search query understanding and automated sentiment analysis.
6. K-Means Clustering Unsupervised Tabular Data Partitions unlabeled data into 'K' distinct groups by calculating the spatial distance between data points. Market segmentation, grouping e-commerce customers by purchasing behavior.
7. PCA
(Principal Component Analysis)		 Unsupervised Tabular Data A dimensionality reduction technique that compresses datasets down to the most critical features while retaining core variance. Compressing images before applying predictive models.
